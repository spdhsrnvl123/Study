TypeScript 설정 파일 = 규칙서
TypeScript는 "이 코드 어떻게 검사할래?" 하는 규칙이 필요하다.

tsconfig.app.json
내가 만드는 웹사이트 코드용 설정
사용자가 크롬 브라우로 보는 화면
일반코드(App.tsx, Button.tsx 등) 브라우저에서 돌아감

tsconfig.node.json
Vite빌드 설정 파일
내 컴퓨터에서 코드를 준비하는 설정
Vite설정(vite.config.ts) 컴퓨터(Node.js)에서 돌아감

noUnusedLocals
선언만 하고 안쓰는 변수 경고

ex)
function example() {
  const name = "구피";  // 에러 - 선언만 하고 안 씀
  const age = 25;      // OK, 아래서 사용함
  
  console.log(age);
}


noUnusedParameters
함수 파라미터 받고 안쓰는 것 경고

ex)
function greet(name: string, age: number) {
  console.log(name);  // age는 안 씀
  // 에러 - age 파라미터를 받았는데 안 쓰고 있어요
}
